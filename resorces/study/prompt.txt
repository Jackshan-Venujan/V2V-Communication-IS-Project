PROJECT CONTEXT
This repository (V2V-Project) is a Vehicle-to-Vehicle (V2V) communication
security simulation written in Python. Two simulated cars exchange "Basic Safety
Messages" (BSMs) over an untrusted network while guaranteeing confidentiality,
integrity, authentication, non-repudiation, forward secrecy, and replay
protection. Attack scripts then prove the defences work. Modules:
- ca/ca.py, ca/issue_cert.py, ca/verify_cert.py — Public Key Infrastructure (CA)
- crypto/crypto.py — cryptographic core (AES-256-GCM, ECDH, HKDF, ECDSA)
- protocol/auth_protocol.py — 4-step mutual authentication handshake
- protocol/v2v_protocol.py — sign-then-encrypt secure messaging (BSM)
- node/vehicle_node.py — TCP networking, the running vehicle program
- dashboard/app.py, dashboard/templates/dashboard.html — Flask monitoring UI
- attacks/replay_attack.py, attacks/tamper_attack.py, attacks/mitm_attack.py
- tests/evaluate_security.py — security evaluation/benchmarks
- config.py — shared constants; readme.md + resorces/ — documentation
